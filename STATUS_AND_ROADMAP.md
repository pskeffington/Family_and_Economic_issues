# Status and Roadmap

## Current status

**Submitted / under journal peer review.**

This repository supports the NLSY79 family-economics and fertility manuscript. The manuscript has been submitted for publication and is currently under journal peer review. The repository should be treated as a reproducibility and audit-control workspace rather than a loose analysis folder.

## Public-facing status language

Use the following wording in CV, portfolio, and public research-status outputs:

> NLSY79 family-economics and fertility manuscript submitted for publication and currently under journal peer review, supported by fixed-effects, count-model, survival-analysis, LaTeX, and table-audit workflows.

## Near-term roadmap

1. Preserve the exact submitted manuscript state.
2. Keep model outputs, generated tables, and LaTeX manuscript tables in parity.
3. Maintain a reviewer-response workspace without changing the submitted baseline.
4. Record any editor or reviewer requests in a dated response log.
5. Prepare a revision branch only after formal journal feedback arrives.

## Validation gates

- Do not replace submitted manuscript files without preserving the submitted snapshot.
- Do not change table values without regenerating and auditing the source model output.
- Keep citation checks and model-table parity documented before any revision submission.

Highest priority among manuscript repositories because it is already in the journal-review pipeline.
