# Project Description

This repository contains the reproducibility and audit-control workspace for a submitted NLSY79 manuscript on family economics, financial preparedness, birth-control education, fertility, and family formation. The manuscript has been submitted for publication and is currently under journal peer review.

The repository is organized to support journal-facing reproducibility rather than exploratory notebook work. Its core purpose is to preserve analysis scripts, model-specific samples, generated tables, citation controls, LaTeX manuscript files, and audit artifacts so that empirical claims remain traceable from data extract to manuscript table.

## Short description

Submitted NLSY79 family-economics and fertility manuscript repository with reproducible R/LaTeX workflows, model audits, and journal peer-review status.

## Scope boundary

This repository should preserve the submitted manuscript state and only support revision work through documented branches or response logs after formal journal feedback arrives.
