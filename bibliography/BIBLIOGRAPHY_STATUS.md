# Bibliography Status

## Current status

Submitted manuscript bibliography must be preserved as part of the journal-review snapshot.

## Rules

1. Do not replace or rewrite submitted citation keys without preserving the submitted manuscript state.
2. New citations for revision should be staged separately until formal reviewer feedback arrives.
3. Every manuscript claim added during revision should map to a BibTeX key and a response-log item.
4. Keep data-source citations, methods citations, and substantive literature citations separated in notes.

## Bibliography workstreams

| Workstream | Status | Notes |
|---|---|---|
| Submitted bibliography snapshot | Preserve | Tie to submitted manuscript version |
| Reviewer-requested citations | Pending | Add only after formal feedback |
| Data-source citations | Audit | Confirm NLSY79/public-use data citation format |
| Methods citations | Audit | Fixed effects, count models, survival analysis, and robustness checks |
| Substantive literature | Audit | Family economics, financial preparedness, reproductive education, fertility |

## Revision checklist

- Citation key exists in BibTeX or bibliography source
- In-text citation resolves in LaTeX build
- Claim supported by cited source
- Table/model claim has matching analytic output
- Any added citation is listed in reviewer-response notes if reviewer-driven
