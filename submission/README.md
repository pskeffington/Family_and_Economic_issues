# Submission Support Files

This directory is for journal-submission materials kept separate from the blinded manuscript.

Blinded manuscript source:

```text
manuscript/NLS-79.tex
```

Recommended final packaging files:

```text
submission/title_page.tex
submission/cover_letter.md
submission/author_disclosures.md
submission/submission_checklist.md
```

Generated upload folders, ZIP archives, PDFs, Word exports, and LaTeX build artifacts should be treated as local packaging output rather than source files. Rebuild them from the tracked manuscript and support files when needed.

Keep author names, affiliations, correspondence details, acknowledgments, and author-specific disclosures outside the blinded manuscript source in `manuscript/NLS-79.tex`.
