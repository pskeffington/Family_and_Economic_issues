# Technical-Check Return Roadmap

## Editorial request

The editor requested one narrow technical correction:

> Please remove the author declaration statement other than funding statement.

The technical-check return must not introduce substantive manuscript changes.

## Exact return scope

The revised manuscript return should include:

1. A blinded manuscript file with author declaration material removed.
2. A separate funding-statement page, outside the blinded manuscript file.
3. A short response-to-technical-check letter confirming the requested correction.

## Files prepared for return

- `manuscript/NLS-79.tex`
- `manuscript/NLS-79.pdf`
- `submission/funding-statement.tex`
- `submission/Funding-Statement-Separate-Page.pdf` generated by workflow
- `submission/editor-response-technical-check.tex`
- `submission/Response-to-Technical-Check.pdf` generated by workflow

## Guardrails

Do not change manuscript title, abstract, results, tables, citations, model estimates, discussion, limitations, conclusion, or references for this technical-check return.

Do not include declarations in the blinded manuscript file. Do not include author contributions, acknowledgments, competing interests, ethics approval, consent statements, or data-availability statements in the revised manuscript file for this return.

Keep the funding statement separate from the blinded manuscript file.

## Funding statement location

Funding statement text is maintained only on the separate funding page:

> No funding was received to assist with the preparation of this manuscript.

## Workflow artifacts

Use the existing editor-response workflow artifact for the response letter and funding page:

- `JFEI-editor-response-technical-check`

Use the existing blinded manuscript workflow artifact for the revised manuscript package:

- `JFEI-blinded-review-package`

## Completion standard

This technical-check return is complete when the submitting author uploads the revised blinded manuscript and separate funding statement through the Springer Nature submission system, with the response letter included only if the system allows or requests a response document.
